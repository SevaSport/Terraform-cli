# Быстрая настройка VPS сервисов

## Цель
Скрипт позволяыет значительно ускорить процесс первичной настройки VPS.
Он выполняет:
- обновление установленных пакетов
- устанавливает базовыет программы из vps.packages[] (config.yml)
- создает дополнительных пользователей без root доступа
- устанавливает базовые программы для обеспечение безопасности VPS: ufw и fail2ban
- устанавливает Docker
- устанавливает специальные программы для VPN: Outline, OpenVPN

## Поддерживаемые операционные системы
### Для VPS
- Ubuntu 20+

### Для управляемого компьютера (на котором запускается скрипт)
- MacOS
- Windows 10+
- Ubuntu 20+


## Способ использования
Перед использованием необходимо скачать удаленный репозиторий, сделать исполняемым файлов run.sh.
Если необходимо можно внести изменения - указав логин и пароль к удаленной машине в credentials.yml , набор устанавливаемых программ и их настройки в config.yml.
Для запуска необходимо выполнить ./run.sh

## Принцип работы скрипта
### Этап 1. Предворительная проверка операционной системы и имеющихся пакетов
- проверка типа операционной машины на управляемой машине
- проверка наличия необходимых пакетов (ssh client, sshpass, yq)
- установка необходимых пакетов, если необходимо

### Этап 2. Проверка конфига для подключения к удаленной машине
- Проверка, указан ли IP-адрес. Если нет, то запрашивается отдельно через консоль.
- Проверка, указан ли PORT SSH. Если нет, то запрашивается отдельно через консоль.
- Проверка, указан ли SSH пароль. Если нет, то запрашивается отдельно через консоль.

### Этап 3. Пробное подключение к удаленному серверу
- Пробное подключения к удаленному серверу
- Поиск публичного ключа на локальной машине, если нет создается
- Поиск публичного ключа удаленного сервер с указанным IP в known_hosts, удаление если он был найден
- Копирование публичного ключа локальной машины но удаленную
- Проверка авторизации по этому ключу

### Этап 4. Первоначальная настройка удаленной машины
- Обновление пакетов
- Установка рекомендуемых базовых пакетов vps.packages[]

### Этап 5. Создание учетных записей
- Создание записей по списку vps.users[]
- Вывод информации о созданных пользователях

### Этап 6. Обеспечение информационной безопасности
- Отключение root пользователя для ssh
- Изменение порта для ssh, если необходимо
- Подключение как пользователь ssh
- Базовая настройка межсетевого экрана
- Базовая настройка fail2ban

### Этап 7. Установка Docker, Docker compose
- Скачивание и установка Docker Engin
- Вывод версии Docker
- Вывод версии Docker Compose
